# Documentation

## Work

`<Work />` is an IIIF viewer that can be used to display digital IIIF manifests. It is built with [Clover-IIIF](https://github.com/samvera-labs/clover-iiif).

### Basic usage

```tsx copy
import { Work } from '@/components/work'

<Work manifest="ubb-diplom-0034">
  <Work.Content>
    <Work.Description>
      This is an example of how you can display a digital object in your exhibition. 
      Replace this text with your own description of the object you want to show.
    </Work.Description>
    <Work.Link href="https://marcus.uib.no/instance/charter/ubb-diplom-0034.html">
      See more in Marcus
    </Work.Link>
  </Work.Content>
</Work>
```

### Direct IIIF URL

```tsx copy
<Work manifest="https://api.ub.uib.no/items/ubb-diplom-0034?as=iiif">
  <Work.Content>
    <Work.Description>
      Dette diplomet er datert 7. august 1348 og handler om et drap i Bergen samme vår.
    </Work.Description>
  </Work.Content>
</Work>
```

### Manifest prop

`manifest` prop can be:

- **String ID**: `"ubb-diplom-0034"` - automatically converted to UB API URL
- **Direct URL**: `"https://api.ub.uib.no/items/ubb-diplom-0034?as=iiif"`
