# Starter Markdown page

You can copy and paste the following code to create a new Markdown page. When you are in the Codespace editor, you can right click on a folder you want to create a new page in and select "New file". Give the file a name (using `-` for spaces, all lowercase) with `.mdx` extension and paste the code into the file.

```tsx copy
---
title: Starter Markdown page
keywords: 
  - markdown
  - page
  - starter
  - markdown page
---
```

## Frontmatter

The frontmatter is the metadata for the page. It is used to set the title, description, keywords, and other metadata for the page. It must be the first thing in the file and must be wrapped in `---` lines.

```tsx
---
title: Starter Markdown page
description: This is a description of the page
keywords: 
  - markdown
  - page
  - starter
  - markdown page
---
```
